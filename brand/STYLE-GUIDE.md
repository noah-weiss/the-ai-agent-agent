# Brand Style Guide · The AI Agent Agent

Version 1.0 · 2026  
Visual board: `brand/style-guide.html`  
Live site: `site/index.html`

## 1. Essence
Prestige agency discipline. CAA restraint. Our hand.

**Promise:** I make AI your secret weapon.  
**Offer:** Results-driven agent. One session. One change.  
**Posture:** Full-service. Just me. Opportunity first.

## 2. Logo
### Primary
Noah handwriting wordmark (`brand/handwriting/crops/01-hero-signature-top.jpg`)  
Hero scale only. Never stretch. Never decorate.

### Institutional
`THE AI AGENT AGENT`  
1px green rule  
`Results-driven agent. One session. One change.`  
`LOS ANGELES`

Use institutional lockup for small UI, docs, legal, nav.

## 3. Color
| Name | Hex | Use |
|------|-----|-----|
| White | #FFFFFF | Field |
| Off-white | #F7F7F7 | Bands |
| Black | #000000 | Type / primary chrome |
| Muted | #666666 | Secondary type |
| Line | #E6E6E6 | Rules |
| Green | #0B6B3A | Authority accent |
| Green deep | #08522C | Hover |

**Rule:** green is a decision, not a theme.

## 4. Type
Stack: Inter, Helvetica Neue, Helvetica, Arial  
No decorative marketing serifs on the site.  
Handwriting is the only expressive type object.

## 5. Layout
- Max width ~1120px
- Sticky sparse header
- Huge vertical space
- 1px rules
- Sharp buttons (no pill soup)
- One primary CTA per view

## 6. Voice
Direct. Confident. Short. Service-first.  
Ban: authentic, seamless, revolutionary, leverage, utilize, AI-powered, em dashes, fake proof.

## 7. Imagery
Empty field + type + handwriting artifact.  
No robots. No neural nets. No startup stock.

## 8. SEO
- Title: Brand | Benefit | City
- One H1
- FAQ
- JSON-LD ProfessionalService
- Canonical + OG + sitemap + robots
- Alt text on handwriting always

## 9. Applications
| Surface | Treatment |
|---------|-----------|
| Site | White prestige, hand hero |
| LinkedIn | Institutional lockup + green rule |
| Session docs | Letterhead, black type, green rule only |
| Favicon | Black field, green bar |

## 10. Quality bar
If it looks like a SaaS template, cut.  
If it looks like “AI brand kit,” cut.  
If CAA would be embarrassed by the clutter, cut.
