# Agency inspiration — CAA · UTA · Roc Nation

Sources checked: caa.com, unitedtalent.com, rocnation.com (2026-07-15)

## What each one owns

### CAA
- Prestige through **restraint**
- Language of **opportunity** and **representation**, not “services”
- Clean, institutional, almost cold confidence
- Practice areas as quiet power, not product tiles
- World-class clients implied, never screamed

Steal:
- Short, serious sentences
- Dark / neutral fields, huge negative space
- Never pitchy. Never “AI-powered”
- Presence > features

Kill if copying:
- Giant multi-office flex
- Endless department taxonomy on day one

### UTA
- Line: **“Uniting ideas, opportunities, and talent”**
- Future-facing without startup cosplay
- “Premier global agency for talent and brands”
- Partner-for-growth posture
- Modern modular site: lockup title → motion/proof → one paragraph thesis

Steal:
- One killer lockup line at the top
- Video/sizzle optional later; thesis first
- Footer as identity stamp (name + city + one line)

### Roc Nation
- **Full-service** as identity, not a package name
- Culture / power / empire energy
- Founder gravity without smallness
- Black, bold, logo-forward
- Entertainment company, not a vendor

Steal:
- Full-service agent language (already ours)
- Black field + one unmistakable mark
- Status without explaining yourself to death

## Visual rules from the inspo
1. **Black is the suit.** Not charcoal soup with 8 greys.
2. **One mark owns the page.** Your hand, or hand + plaque. Not icon salad.
3. **Type hierarchy like an agency deck:** one huge line, one supporting, stop.
4. **Motion is optional.** Stillness is power.
5. **Nav stays tiny.** Work / How / Contact. That’s it.
6. **No stock people.** No robot. No purple neural nets.
7. **City + domain in the chrome** like an agency letterhead.

## Copy rules from the inspo
- Prefer “represent / partner / opportunity / full-service”
- Avoid “solutions / platform / leverage / seamless”
- One thesis, repeated with discipline
- Let the name do heavy lifting
