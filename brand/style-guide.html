5. Components

Buttons
Primary black Primary green Ghost

Sharp corners. No pill soup. Height 44. Weight 600. One primary action per view.

Rules / space

1px rules. Huge margins. Content max ~1080px. Sections breathe. If it looks busy, cut half.

6. Voice

Sound like

  • Direct
  • Confident without posing
  • Service-first
  • Short lines
  • Opportunity language

Never

  • authentic / seamless / revolutionary
  • AI-powered / leverage / utilize
  • em dashes
  • fake case studies
  • startup hype cadence

7. Imagery

Prefer empty field + type + one artifact (handwriting). Photography only if editorial and restrained. No robots. No neural-net stock. No founders-in-beanbags.

8. Do / Don’t

Do

  • White field, black type
  • One green decision per screen
  • Handwriting at heroic scale
  • Agency letterhead discipline
  • Cut copy until it hurts a little

Don’t

  • Soft UI cards with glow blobs
  • Pill badges and check lists as decoration
  • Trendy display serifs everywhere
  • Dark mode as default brand
  • Generic AI visual language

9. SEO standards

  • Title: Brand | Benefit | City
  • One H1. Logical H2s. FAQ for long-tail.
  • JSON-LD ProfessionalService
  • Canonical, OG, sitemap, robots
  • Alt text on handwriting mark always
