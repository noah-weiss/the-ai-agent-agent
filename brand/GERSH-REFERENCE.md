# Gersh reference · applied

Source: https://gersh.com/

## What makes Gersh cool
- Almost no homepage copy noise
- Fixed letterhead + Menu
- Full-viewport intro
- Giant category index (Talent / Literary / etc.)
- Full-color identity field (their blue, our green)
- Neue Montreal energy: clean grotesk, weight 400–500 headlines
- Huge type, huge space, thin rules
- Menu becomes a second page in brand color

## What we stole
1. Fixed header + Menu panel in brand green
2. Full-height intro with signature logo as the mark
3. Giant practice index list (Session / Concierge / Build / Curation)
4. Green full-bleed statement band (their blue field)
5. Satoshi type (closest free Neue Montreal vibe)
6. Sparse hero: one line + one support line

## What we kept
- Dad language (placed in About / practice detail, not dumped in hero)
- Green as identity color (#0B6B3A)
- Handwriting signature as personal agent mark
- Request representation CTA
