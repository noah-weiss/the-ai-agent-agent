# Handwriting capture

This is the brand spine. Not a script font. Your hand.

## Gear
- White paper (printer paper is fine)
- Black pen you actually like (gel or fine Sharpie)
- Phone camera
- Flat desk, overhead angle, good light, no heavy shadow

## Process (10 minutes)
1. Write each line **3× at natural speed**
2. Write each line **1× slower and cleaner**
3. Optional: one big messy version that still reads
4. Photograph each page flat
5. Drop photos in Hermes chat **or** Notion page: **01 — Handwriting Capture**
