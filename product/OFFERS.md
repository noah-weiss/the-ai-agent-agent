# Offers (v0 draft)

Three packages. No buffet. No fake tiers for the sake of tiers.

## 1. Agent Design Sprint

**What:** Map one painful workflow. Decide what an agent should own, what a human must keep, and what good looks like when it fails.

**You leave with:**
- process map
- agent brief (tools, memory, handoffs, success metrics)
- build/no-build recommendation
- rough build estimate

**Time:** ~1 week  
**Fit:** "We want agents but do not know where to start."

**Price draft:** $2,500–$4,000 fixed  
*(finalize after first two sales conversations)*

## 2. Agent Build

**What:** Design + stand up one production-ready agent (or small multi-agent workflow) for a scoped job.

**You leave with:**
- working agent
- runbook
- failure modes documented
- handoff training for your team
- optional one-week stabilization

**Time:** 2–4 weeks depending on tools/APIs  
**Fit:** Clear workflow, real data access, owner on your side.

**Price draft:** $6,000–$15,000 fixed scope  
*(quoted after Design Sprint or paid discovery)*

## 3. Agent Ops

**What:** Keep the agent useful after launch. Monitoring, prompt/tool updates, weekly review, on-call for breakage.

**You leave with:**
- monthly reliability notes
- change log
- priority queue for improvements

**Time:** monthly  
**Fit:** Agent is live and tied to revenue or critical ops.

**Price draft:** $1,500–$4,000 / month  
*(based on blast radius and hours)*

## Rules of engagement

- Paid discovery or Design Sprint before large builds.
- No open-ended "AI strategy" retainers with no ship date.
- Client must provide a human owner and tool access.
- We refuse work that should stay human (judgment, relationship repair, legal advice).
- Boardroom test on every deliverable.

## Sample agent ideas (dogfood or demo)

1. **Intake + triage agent** for consulting leads (qualify, draft reply, book or park)
2. **Ops capture agent** (messy notes → structured tasks + weekly brief)
3. **Agent-native CLI wrapper** for a client's existing API (Printing Press path)
